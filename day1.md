# Day 1 - Linux Basics

## What is Linux?
Linux is an open-source operating system used in servers, cloud platforms, and DevOps environments.

## Commands I Learned Today
- pwd → shows current directory
- ls → lists files
- cd → change directory
- mkdir → create folder
- touch → create file
- rm → delete file

## What I Practiced
- Created folders
- Created files
- Deleted files
- Navigated directories
